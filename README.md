Программа реализует конечный автомат (машину состояний) на языке Structured Text для CODESYS.

Автомат имеет 4 состояния: 1 → 2 → 3 → 99 (ошибка). Переключение между состояниями происходит по кнопке «Дальше» (F1). Из состояния ошибки выход только по кнопке «Сброс» (F2).

Каждому состоянию соответствует своя лампочка (light1, light2, light3, lightErr), которая загорается при входе в состояние.

Ключевые особенности:

- Детекция фронта нажатия кнопки (однократная реакция)

- Защита от невалидных состояний (блок ELSE)

- Правильная инициализация переменных
